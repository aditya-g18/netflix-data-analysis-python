# Netflix Data Analysis – Python

## 📌 Project Overview
Performed exploratory data analysis (EDA) on Netflix's content dataset (8,807 titles)
to uncover trends in content type, country distribution, and release patterns.

## 🛠️ Tools Used
- Python (Pandas, NumPy, Matplotlib)

## 📊 Key Insights
- Netflix has 8,807 titles — 69.6% Movies, 30.4% TV Shows
- United States is the top content-producing country (2,818 titles)
- India ranks 2nd with 972 titles
- Content releases grew significantly after 2015

## 📁 Files
- Netflix_Datasets.ipynb — Main analysis notebook
- Netflix_Datasets.csv — Dataset file

## ▶️ How to Run
1. Clone this repo
2. Install requirements: pip install pandas numpy matplotlib
3. Open and run Netflix_Datasets.ipynb in Jupyter Notebook
